# Projeto de Coordenação de Horários

Sistema automatizado para geração de grades de horários acadêmicos, desenvolvido para a disciplina de Software Engineering.

## Objetivo
O sistema visa automatizar a alocação de professores, turmas e salas, eliminando conflitos de horários e respeitando restrições de capacidade e tipo de sala (Teórica/Laboratório).

## Tecnologias Utilizadas
Linguagem: Python
Biblioteca Principal: Pandas (Manipulação de dados CSV)
Versionamento: Git & GitHub
